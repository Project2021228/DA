Поиск основания систем счисления.

1. Укажите через запятую в порядке возрастания все основания систем счисления, в которых запись числа 22 оканчивается на 4.

  • filter_drama
    Пример решения:
    Для начала нужно найти все целые числа N > 4 (т.к. последней цифрой числа должна быть 4) такие, что статок от деления 22 на N равен 4
    Следовательно получаем уравнение: 22=k*N+4, где k - целое неотрицательное числи(0, 1, 2, ...);
    Из формулы получаем, что k*N=18 ,так что задача сводится к тому, чтобы найти все делители числа 18, которые больше 4;
    В этой задаче есть только три таких делителя: 6, 9, 18.

    Ответ: 6, 9, 18.


2. В системе счисления с некоторым основанием десятичное число 129 записывается как 1004. Укажите это основание.

  • filter_drama
    Пример решения:
    Для начала нужно найти все целые числа N > 4 (т.к. последней цифрой числа 1004 является 4) такие, что статок от деления 129 на N равен 4;
    Обозначим за х нужную систему счисления, тогда будет выполняться тождество(перевод из системы счисления с основанием х в десятичную):
    x3+4*x0=129
    x3+4*1=129
    x3=125
    ∛125 = 5
    Следовательно x = 5

    Ответ: 5.


3. Укажите наименьшее основание системы счисления, в которой запись числа 70 трехзначна.

  • filter_drama
    Пример решения:
    Обозначим через N неизвестное основание системы счисления, тогда запись числа 70 в этой системе имеет вид:

    x * y * z N = 70,

    Вспомним алгоритм перевода числа из системы счисления с основанием N в десятичную систему: расставляем сверху номера разрядов и умножаем каждую цифру на основание в степени, равной разряду:
    210
    xyz N = x * N2 + y * N + z = 70;
    Поскольку запись трехзначная, x ≠ 0, поэтому 70 ≥ N2;
    С другой стороны, четвертой цифры нет, то есть, в третьем разряде – ноль, поэтому 70 < N3;
    Объединяя последние два условия, получаем, что искомое основание N удовлетворяет двойному неравенству:

    N2 ≤ 70 < N3

    Учитывая, что N – целое число, методом подбора находим целые решения этого неравенства; их несколько – 5, 6, 7, 8:

    52 = 25 ≤ 70 < 53 = 125

    62 = 36 ≤ 70 < 63 = 216

    72 = 49 ≤ 70 < 73 = 343

    82 = 64 ≤ 70 < 83 = 512

    Минимальное из этих значений – 7

    Ответ: 7.
